# Word Guessing Game (Python)

A simple command-line word guessing game built using Python. 
The player tries to guess a hidden word by entering letters.

## Features
- Random word selection
- User input handling
- Game logic using loops and conditions
- Displays correct and incorrect guesses

## Technologies Used
- Python

## How to Run
1. Download the project.
2. Run the file:

python word_game.py
